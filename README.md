## How It Works

- The game presents a 3×3 grid.
- Players alternate clicking empty cells, placing **X** first, followed by **O**.
- A helper function evaluates the board after each move to detect a win (three in a row vertically, horizontally, or diagonally) or a draw.
- On game completion, an alert notifies the result, and the board is reset for a new round.

---

## Gameplay Instructions 

1. Player 1 clicks an empty cell to place **X**.
2. Player 2 follows by placing **O** in an empty cell.
3. Continue alternating until either:
   - One player wins with three in a row.
   - Or all cells are filled, resulting in a draw.
4. A popup announces the outcome, and the game is reset automatically for another round.
